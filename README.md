# Overview

BarberShop API is the backend service for a modern Full-Stack barber shop management system.

The API provides business logic, authentication, database access and RESTful endpoints consumed by the BarberShop Frontend application.

This project demonstrates backend development using ASP.NET Core, SQL Server and Docker while following modern API design principles.

# Running the Project

Clone the repository

```bash
git clone https://github.com/EduardoHAnacleto/BarberShop.git
```

Run with Docker

```bash
docker compose up --build
```

Open Swagger in your browser to explore and test the API.

For the best evaluation experience, I recommend running the complete Full-Stack solution available in the BarberShop-Front repository, which includes the frontend, backend, database and Docker configuration ready for local deployment.
